Staffy — HR-платформа для ресторанов

Staffy — это веб-приложение для управления персоналом в ресторанах и кафе, которое помогает снижать текучесть сотрудников, ускорять адаптацию и улучшать качество сервиса.

🎯 Проблема

Рестораны теряют деньги из-за:
- высокой текучести персонала
- долгой адаптации новых сотрудников
- отсутствия контроля обучения
- слабой аналитики по персоналу

💡 Решение

Staffy помогает:
- быстро обучать новых сотрудников
- отслеживать прогресс адаптации
- управлять сменами
- анализировать риск выгорания и текучести

🧭 Пользовательский сценарий

Главная → Регистрация → Обучение → Личный кабинет → Панель менеджера → Тариф → Успешная оплата

👥 Роли пользователей

- Официант
- Бармен
- Хостес
- Менеджер ресторана

⚙️ Функционал

- Онбординг сотрудников
- Обучающие модули
- HR-аналитика
- Воронка кандидатов
- Статусы сотрудников (зелёный / жёлтый / красный)
- Тарифная система


🧪 Демо

Проект создан как MVP для демонстрации на защите

🛠 Технологии

- React / Vite (или Replit environment)
- UI компоненты
- Mock-данные
- Клиентская логика без backend

📊 Данные

Используются реалистичные тестовые данные сотрудников и аналитики для демонстрации бизнес-логики.

🚀 Цель проекта

Показать полный end-to-end HR сценарий для ресторанного бизнеса.
